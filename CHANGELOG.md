# Change Log

All notable changes to this project will be documented in this file.

## 0.0.1

- Initial Marketplace release.
- Inline PowerShell evaluation for the current line, selection, or full file.
- Persistent PowerShell session with selectable `pwsh` or `powershell` engine.
- Structured inline previews, dedicated result preview panel, and output channel support.
- Session restart command and engine status in the status bar.
